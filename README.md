# MessMate

MessMate is a smart hostel mess management prototype focused on:

- Meal Intent Tracking
- Live Crowd Monitoring
- Complaint Submission and Tracking

## Tech Stack
- React
- Vite
- Tailwind CSS
- LocalStorage

## Features
- Name-based quick entry
- Personalized dashboard
- Meal response form
- Complaint form with saved entries
- Live crowd status page

## Run Locally
npm install
npm run dev

## Build
npm run build
